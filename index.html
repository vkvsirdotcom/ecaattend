📱 Updating the Android APK
1. Build new APK in Android Studio
2. Replace app.apk on Netlify (same filename)
3. Update app version above → Save app links
4. All users get update prompt on next app launch
🌐 Updating the web app
Edit index.html → drag onto Netlify → live in 30s. PWA users auto-update on next open.
⚙️ Updating the Apps Script
Open script → paste new code → Save → Deploy → Manage deployments → pencil icon → New version → Deploy. Same URL.
👥 Adding teachers
Paste Name | Email | Branch | Abbr lines. Sheets auto-created in your Drive → ECA Attend System → Teacher Sheets.
❌ Common errors
"Script returned plain text": Old version deployed. Re-deploy with "New version".
"Session not found": Code expired. Reopen session.
"Sheet ID not set": Teacher sheet not created — re-add teacher.
"Connection failed": Check Apps Script URL.
